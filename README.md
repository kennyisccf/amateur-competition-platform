# amateur-competition-platform
综合业余比赛平台
先问大家一个问题：你身边有没有这样的朋友？他周末喜欢打篮球，水平还不错，想找点正规比赛打打，但市面上的赛事要么职业门槛太高，要么是野球局，打完就散，没有任何记录。他想知道自己在本地业余篮球圈到底是什么水平，但没有答案。再问一个问题：如果你和几个朋友想进行一场小型的棋牌赛，你用什么工具组织记录？ 
这就是我们开发这款软件的动机。
我们发现，中国有庞大的业余爱好人群，但在“专业职业赛事”和“娱乐约局”之间，存在一个巨大的服务断层。现有的软件要么只做单一运动；要么只做报名工具；要么根本没有成体系的积分和档案管理。我们要做的，就是填补这个断层，打造一个真正服务于“全民业余比赛”的标准化平台。
